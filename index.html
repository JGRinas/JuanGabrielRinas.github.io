Calculadora simple

HardGame-Ecommerce de productos informáticos

Proyecto de la materia taller de programación 2. Para la creación de esta página se utilizaron los lenguajes: HTML, CSS con bootstrap, JavaScript y PHP con el framework CodeIgniter5, también una base de datos relacional, gestionada en phpmyadmin. Se trata de una página ecommerce de productos informáticos. Cuenta con dos tipos de usuarios: Cliente y Administrador. El cliente puede ver y comprar los productos disponibles, enviar consultas a los administradores de la página y gestionar el perfil de su cuenta. Los administradores por otra parte, pueden registrar y gestionar productos, ver y responder las consultas enviadas y gestionar usuarios y ventas. Ir al repositorio de github

Cambiar el color del fondo por voz

Pequeño proyecto para practicar comandos por voz. Se utilizó únicamente HTML, CSS y JavasCript utilizando la API SpeechRecognition. Primero debe habilitar el micrófono y luego decir alguno de los siguientes colores: Rojo, azul, verde, amarillo y naranja. Si se dice otra palabra que no sea alguna de ellas, el color del fondo no se cambiará pero se mostrará la palabra dicha. Click en la imágen para ir a la página

FileTable-Galeria de archivos

Galería de archivos realizada con c# winform, con una base de datos en SQL Server. Para el manejo de archivos multimedia se utilizó una tabla de usuario especializada, conocida como FILE TABLE. Para la creación del programa se tuvo en cuenta la creación de una Vista en la base de datos, en la cual se convirtió el tipo de dato HierarchyID del File table en un varchar(), debido a que el tipo de dato HierarchyID no es reconocido por las consultas a la base de datos dentro de la aplicación, generando errores en las mismas. En la aplicación se pueden ver todos los archivos que están dentro del directorio FILE TABLE en la base de datos, así como ver los archivos que son solo imagenes, videos y audios por separado. Ir al repositorio de github

SpaceShip-Juego de consola de windows

Para la cración de este juego se utilizó unicamente C++ y conceptos de Programación Orientada a Objetos. Al principio se debe escribir la dificultad en la que se quiere jugar. La nave cuenta con 3 vidas, cada una con 3 colisiones antes de explotar y perder una vida, al perder las 3 vidas terminará el juego mostrando la cantidad de puntos que se hicieron durante partida.
