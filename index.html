Galeria de archivos

Galería de archivos hecho con c# winform, con una base de datos de SQL Server, para el manejo de archivos se utilizó una tabla de usuario especializada, conocida como FILE TABLE Para la creación del programa se tuvo en cuanta la creación de una Vista en la base de datos, en la cual se convirtió el tipo de dato HierarchyID del File table en un varchar(), debido a que el tipo de dato HierarchyID no es reconocido por las consultas a la base de datos dentro de la aplicación, generando errores en las mismas. En la aplicación se pueden ver todos los archivos que están dentro del directorio FILE TABLE en la base de datos, así como ver los archivos que son solo imagenes, videos y audios por separado. Ir al repositorio de github

SpaceShip: Juego en consola de windows

Juego en consola de windows utilizando unicamente c++. Al principio se debe escribir la dificultad en la que se quiere jugar. La nave cuenta con 3 vidas, cada una con 3 colisiones antes de explotar y perder una vida, al perder las 3 vidas terminará el juego mostrando la cantidad de puntos que se hicieron en la partida.
